This project contains a set of python codes that use natural language processing techniques to analyze and mine text data.

Installation
To run this code, you will need to install the following Python packages:

NLTK
Pandas
Numpy
Matplotlib
requests
bs4
openpyxl

You can install these packages using pip:

Copy code
pip install nltk pandas numpy matplotlib bs4 requests openpyxl
Usage
To use this code, follow these steps:

Open the Web.py file.
Specify the path to the text file you want to analyze.
Run the script.
The output will include a table showing all variables about the text, such as the negative and polarity score.
